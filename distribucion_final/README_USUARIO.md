# Sistema de Inventario Desktop v1.0.0

## 📦 Instalación

### Opción 1: Instalador Recomendado
1. Descarga el archivo `inventario_desktop_0.1.0_x64-setup.exe`
2. Haz doble clic para ejecutar el instalador
3. Sigue las instrucciones del asistente
4. La aplicación se instalará en Program Files

### Opción 2: Ejecutable Portable
1. Descarga el archivo `inventario_desktop_app.exe`
2. Haz doble clic para ejecutar directamente
3. No requiere instalación

## 🚀 Primer Uso

1. **Registra tu primera cuenta:**
   - Abre la aplicación
   - Haz clic en "Registrarse"
   - Completa tus datos (nombre, email, teléfono, empresa)
   - Crea una contraseña segura

2. **Inicia sesión:**
   - Usa tu email y contraseña
   - Serás redirigido al panel principal

## ⚙️ Configuración del Perfil

1. Ve a "Ajustes" o "Perfil"
2. Podrás:
   - Actualizar tu información personal
   - Subir tu foto de avatar
   - Cambiar tu contraseña

## 📋 Funcionalidades Principales

### 🛒 Compras
- Registrar nuevas compras de productos
- Asignar proveedores y fechas
- Control de stock automático

### 💼 Facturación
- Crear facturas para clientes
- Generar números de factura automáticos
- Imprimir y exportar facturas

### 📦 Inventario
- Gestión completa de productos
- Control de stock en tiempo real
- Alertas de bajo stock

### 📊 Reportes
- Movimientos de inventario
- Estadísticas de ventas
- Reportes de compras

## 🔧 Solución de Problemas

### La aplicación no inicia
- Asegúrate de tener Windows 10 o superior
- Verifica que tienes permisos de administrador
- Intenta ejecutar como administrador

### No puedo agregar una compra
- Asegúrate de haber iniciado sesión
- Verifica que tengas productos registrados
- Revisa que todos los campos obligatorios estén completos

### Problemas con la base de datos
- La aplicación crea una base de datos local automáticamente
- Los datos se guardan en la carpeta de instalación
- No requiere configuración de base de datos externa

## 📞 Soporte

Si encuentras algún problema:
1. Revisa que todos los campos estén completos
2. Asegúrate de tener conexión estable
3. Reinicia la aplicación e intenta nuevamente

## 🖥️ Requisitos del Sistema

- **Sistema Operacional:** Windows 10 o superior
- **Memoria RAM:** Mínimo 4 GB (recomendado 8 GB)
- **Espacio en disco:** 100 MB disponibles
- **Procesador:** Intel i3 o equivalente
